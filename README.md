# MineNet: Co-Saliency Detection with Multiple Inducible Co-Attention and Edge Guidance

## Training
```
cd MineNet/
python3 train.py --tmp tmp/MineNet
```

## Testing
```
cd MineNet/
python3 test.py 
```
